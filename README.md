# Valerio

Questo repository contiene un semplice mockup dinamico di una piattaforma B2B.

Per visualizzare l'esempio apri `web-mockup/index.html` in un browser.

Per esplorare il codice sorgente in modo interattivo puoi aprire `web-mockup/code-viewer.html` con un piccolo server locale (ad esempio `python -m http.server`).
